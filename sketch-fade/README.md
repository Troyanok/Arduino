Генерирование ШИМ-сигнала с помощью функции *analogWrite(*, *)*

*analogWrite* на вход получает:
1. Номер контакта, на который нужно подать аналоговый ШИМ сигнал;
2. 8-разрядное двоичное число.

На вывод ШИМ значение от 0 до 2^8 - 1 (0 .. 255). Яркость светодиода изменяется от 0 до 255. 
Если полученное значение будет 256, то диод будет выключен (2^8 == 0 mod 256)
